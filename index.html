JReliability - The Java-based Reliability Library

JReliability allows to derive several reliability-related measures like Mean-Time-To-Failure (MTTF) or Mission-Time (MT) of complex systems that are modeled using Boolean functions.

The library is highly extensible such that special evaluators to derive other measures as well as other system representations can be included seamlessly.

Moreover, JReliability offers a graphical user interface (GUI) for the visualization of the derived reliability measures as well as the possibility to visualize the Boolean representation of the system.

Hosting, Download, and Documentation

JReliability is hosted on Github where you will find the latest builds and releases. This website additionally provides a documentation that covers how to use JReliability, the JavaDoc, as well as a tutorial.

JReliability is powered by:

  • JavaBDD (BDD Library)
  • Ptolemy Plot (Plot Library for Java)

Other non-commercial reliability evaluation tools and libraries:

  • Good Enough Reliability Tool (Software Reliability Metrics)
